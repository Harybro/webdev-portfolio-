Project 1

Weather Dashboard App

Project Overview

A responsive weather dashboard application built with vanilla JavaScript and OpenWeather API. The app displays real-time weather data including temperature, humidity, wind speed, and a 5-day forecast for any city in the world.

Key Features:

  • Search functionality to find weather for any location
  • Displays current weather conditions with animated weather icons
  • 5-day weather forecast with hourly predictions
  • Temperature unit toggle (Celsius/Fahrenheit)
  • Local storage to save favorite cities
  • Responsive design that works on mobile and desktop

Technologies Used:

HTML5, CSS3, JavaScript ES6, Bootstrap 5, OpenWeather API, Local Storage API

Project Details:

Duration: 3 weeks | Status: Completed | Team Size: Solo Project

This project was created to strengthen my JavaScript fundamentals and API integration skills. It taught me about asynchronous programming, working with JSON data, and creating a clean user interface with Bootstrap.
